# Сириус.Аренда (sirius-rental)

Cервис для бронирования переговорных комнат и учебных пространств на федеральной территории «Сириус». Проект разработан для конкурса «Всероссийский ИТ-раунд».

## ––––– Технологический стек –––––

-  **Язык программирования:** Python 3.14, FastAPI 
- **База данных:** SQLite – SQLAlchemy
- **Проверка корректности данных:** Pydantic
- **Сервер:** Uvicorn

## –––– Выполненный Функционал ––––

- **Управление пространствами (CRUD)**
- **Бронирование**
- **Отмена бронирования**
- **Расписание**
- **Обработка ошибок:**
`200 OK`, `201 Created`, `400 Bad Request`, `404 Not Found`, `409 Conflict`.
- **Поиск свободных пространств** 

## 🚀 Инструкция по запуску
```bash
git clone https://github.com/Vorotkos/sirius-rental.git
cd sirius-rental
```
####  Ошибка "ModuleNotFoundError" при запуске?
Если при запуске терминал указывает на отсутствие библиотек, значит, на вашем компьютере ещё не установлены зависимости. Выполните эти команды в папке проекта:
1. `python -m venv venv`
2. `venv\Scripts\activate`
3. `pip install -r requirements.txt`
4. `python main.py`
- если командная строка жалуется на _python_, то попробуйте использовать _py_.
