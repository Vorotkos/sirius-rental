# Сириус.Аренда (sirius-rental)
REST API сервис с веб-интерфейсом для бронирования переговорных комнат и учебных пространств. Проект разработан для конкурса «Всероссийский ИТ-раунд».

## Технологический стек

- **Язык:** Python 3.14, FastAPI
- **База данных:** SQLite – через SQLAlchemy
- **Валидация:** Pydantic V2
- **Веб-интерфейс:** HTML, JavaScript (Bootstrap 5)
- **Сборка в .exe:** PyInstaller

##  Реализованный функционал

### Техническое задание
- **Исполняемый файл:** запуск через EXE файл.
- **Управление пространствами(CRUD):** создание, получение списка, редактирование и удаление комнат.
- **Бронирование:** создание бронирований с автоматической проверкой пересечений по времени.
- **Отмена бронирования.**
- **Расписание:** получение списка бронирований конкретной комнаты на выбранную дату.
- **Обработка ошибок:** HTTP-статусы – `201 Created`, `400 Bad Request`, `404 Not Found`, `409 Conflict`.

### Доп. задачи из ТЗ
- **Веб-интерфейс:** простая графическая оболочка для управления без использования Swagger.
- **Поиск свободных пространств:** эндпоинт `GET /rooms/available`(ещё я сделал разделение на свободные и занятые в выбранный день).
- **Docker-упаковка:** проект собран в контейнер для развертывания.
- **Файл юнит-теста:** проверка на работоспособность кода.

Так же я сделал доп файл с записью логов API для удобства их просмотра.

## Инструкция по запуску:
### Вариант 1 – через .exe файл
Скачать архив по [ссылке](https://github.com/Vorotkos/sirius-rental/releases), разархивировать его и запустить _sirius-rental.exe_.

### Вариант 2 – через терминал
Сперва скачайте зависимости:
1. `python -m venv venv`
2. `venv\Scripts\activate`
3. `pip install -r requirements.txt`

Затем поочерёдно введите в терминал следующие строки:
```bash
git clone https://github.com/Vorotkos/sirius-rental.git
cd sirius-rental
python main.py
```

## Инструкция по открытию программы с Swagger UI:
После открытия сервиса добавьте в конце адреса `/docs`. Должно получиться http://127.0.0.1:8000/docs

## Запуск тестов:
После установки репозитория(https://github.com/Vorotkos/sirius-rental.git) убедитесь, что у вас включено виртуальное окружение `venv312\Scripts\activate` и в терминале введите `py test_main.py -v`.
