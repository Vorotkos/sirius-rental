# Сириус.Аренда (Sirius.Rental)

REST API сервис для бронирования переговорных комнат и учебных пространств на федеральной территории «Сириус». Проект разработан для конкурса «Всероссийский ИТ-раунд».

## ––––– Технологический стек –––––

- **Python 3.14**,  **FastAPI** 
- 
- **База данных:** SQLite (через SQLAlchemy)
- **Валидация данных:** Pydantic
- **Сервер:** Uvicorn

## –––––  Функционал –––––

### Базовые требования (MVP)
- **Управление пространствами (CRUD)**
- **Бронирование**
- **Отмена бронирования**
- **Расписание**
- **Обработка ошибок:**
(`201 Created`, `400 Bad Request`, `404 Not Found`, `409 Conflict`).
-  **Поиск свободных пространств**
