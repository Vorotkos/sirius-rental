# Сириус.Аренда (sirius-rental)
REST API сервис с веб-интерфейсом для бронирования переговорных комнат и учебных пространств. Проект разработан для конкурса «Всероссийский ИТ-раунд».

## Технологический стек

- **Язык:** Python 3.14
- **Фреймворк:** FastAPI
- **База данных:** SQLite – через SQLAlchemy
- **Валидация:** PydanticV2
- **Веб-интерфейс:** HTML5, CSS3, JavaScript (Bootstrap 5)
- **Контейнеризация:** Docker & Docker Compose
- **Сборка:** PyInstaller

##  Реализованный функционал

### Базовые требования (MVP)
- **Управление пространствами (CRUD):** создание, получение списка (с фильтрацией по вместимости и оборудованию), редактирование и удаление комнат.
- **Бронирование:** создание бронирований с автоматической проверкой пересечений по времени.
- **Отмена бронирования:** мягкое удаление (изменение статуса на `cancelled`).
- **Расписание:** получение списка бронирований конкретной комнаты на выбранную дату.
- **Обработка ошибок:** корректные HTTP-статусы (`201 Created`, `400 Bad Request`, `404 Not Found`, `409 Conflict`).

### Доп. задачи из ТЗ
- **Веб-интерфейс:** простая графическая оболочка для управления без использования Swagger.
- **Поиск свободных пространств:** эндпоинт `GET /rooms/available`(так же я сделал разделение на свободные на весь день, и занятые в выбранный день).
- **Docker-упаковка:** проект полностью контейнеризирован для быстрого развертывания.

## Инструкция по запуску
Поочерёдно введите в терминал следующие строки:
```bash
git clone https://github.com/Vorotkos/sirius-rental.git
cd sirius-rental
```
####  При запуске появляется ошибка из-за отсутсвия библиотек?
Если при запуске терминал указывает на отсутствие библиотек, значит, на вашем компьютере ещё не установлены зависимости. Выполните эти команды в папке проекта:
1. `python -m venv venv`
2. `venv\Scripts\activate`
3. `pip install -r requirements.txt`
4. `python main.py`
- #### если командная строка жалуется на _python_, то попробуйте использовать _py_.
